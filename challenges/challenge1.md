Challenge 1
===========

Function prototype:
-------------------
def sum_num(a, b);
 
* a is the first integer input
* b is the second integer input
 
Return values
-------------
sum of the above two integers
 
Example
-------
Sample Input
a = 2
b = 3
 
Sample Output
5
 
Explanation
The sum of the two integers a and b is computed as: 2 + 3 = 5
