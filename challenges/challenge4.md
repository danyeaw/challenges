Challenge 4
===========

Function prototype:
-------------------
def find_magic_index(arr)
 
arr (list): is the sorted array of distinct integers
 
Return values
-------------
magic index (int): the index i in the arrary such that arr[i] == i
 
Example
-------
Sample Input
[-1, 0, 1, 2, 4, 10]
 
Sample Output
4
 
Explanation
arr[4] == 4, and therefore is the magic index
