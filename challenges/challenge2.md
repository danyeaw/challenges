# Challenge 2

### Function prototype:
    def missing_num(arr)
arr (list): an array of 10 unique numbers from 1 to 11, with one missing
 
### Return values
missing (int): the missing integer
 
### Example
##### Sample Input
    [3, 4, 7, 6, 9, 10, 1, 2, 5, 8]
 
##### Sample Output
    11


 
##### Explanation
The missing integer in the array is 11
